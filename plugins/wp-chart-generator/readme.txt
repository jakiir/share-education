This plugin allows you to easily manage graphics on your blog 

== Description ==

Wp chart generator makes your life easier to create graphics on your blog. With a single management in the back office, you can create as many graphics as you want. Management of these charts is based on custom post types.

Create a graphic is simple : 

*   First clic on \"add a new graphics\"
*   Then enter your data
*   Publish it and a shortcode will automatically be generated
*   Finally copy this shortcode and paste it into an article
*   Go to the article and TADA a graphic appeared

<div>
  Enjoy guys with this plugin.
</div>

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
